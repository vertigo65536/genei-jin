>  **%ping:**
>  returns "pong" and confirms bot is active and replying to messages
>  **%co:**
>  returns simpsons/futurama/family guy/rick and morty clip correspondng to quote
>  **%co+:**
>  same as co, but forces the clip to be 20s, starting with quote
>  **%co-:**
>  ends with quote
>  **%co=:**
>  quote is centered
>  **%wiki:**
>  searches wikipedia and retrieves the first page corresponding to the query
>  **%yt:**
>  searches youtube and retrieves the first video corresponding to the query
>  **%gi:**
>  searches google images and retrieves the first video corresponding to the query
>  **%title:**
>  sets the server title
>  **%ctitle:**
>  sets the channel title
>  **%topic:**
>  sets the channel topic
>  **%dex:**
>  searches the pokedex
>  **%lucky:**
>  Returns the "i'm feeling lucky" result of a google search of a given query
>  **%stats:**
>  returns either all stats for user, or if given a param, stats for that search for user
>  **%trophylist / %tl**
>  Returns a list of all trophys
>  **%mytrophys / %mt**
>  Returns all your attained achivements
>  **%trophystat / %ts**
>  Returns all trophies ordered by attained rate
>  **%nick <user>**
>  Sets nick of user
>  **%game:**
>  returns approximate value of a searched game
>  **%yu:**
>  searches for a yugioh card and returns name/description/value
>  **%man:**
>  returns this man page
>  **%help:**
>  alias to %man
